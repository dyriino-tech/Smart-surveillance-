# Smart Surveillance Ecosystem
By Dyriino Tech Limited

## About
A low-cost smart surveillance system using Python, OpenCV, Telegram, and Arduino — combining a camera-based motion detection pipeline with an autonomous LDR streetlight module.

## Files
- `camera_v2.py` — ⭐ LATEST VERSION: Motion detection, photo capture, Telegram alerts with GPS location
- `Camera.py` — Earlier version (basic motion detection)
- `ldr_streetlight.ino` — Arduino sketch for automatic LDR-based streetlight control

## Features
- Live camera monitoring via DroidCam
- Motion detection with visual alerts
- Automatic photo capture
- Telegram alerts with photo and GPS location
- LDR-based automatic streetlight control
- Future: Drone response

## Hardware Used
- Android smartphone (camera)
- Windows laptop
- DroidCam app
- Arduino board with LDR sensor and LED/relay

## Software Used
- Python 3.8
- OpenCV 4.5
- Telegram Bot API
- Arduino IDE

## How to Run
1. Connect DroidCam on phone and laptop
2. Open CMD as Administrator
3. cd Desktop
4. python camera_v2.py
